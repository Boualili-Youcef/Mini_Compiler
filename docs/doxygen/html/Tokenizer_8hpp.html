Description détaillée

Définit les structures et classes nécessaires pour l'analyse lexicale.

Ce fichier implémente un tokenizer qui transforme le code source en une séquence de tokens utilisables par le parser. Il utilise les principes d'informatique théorique pour réaliser l'analyse lexicale.
